# Telegram-бот для приёма заявок

Бот автоматически собирает заявки от клиентов в Telegram и передаёт их менеджеру, а также сохраняет данные в Google Sheets.

## Функционал
- пошаговый сбор данных
- проверка на пустые значения
- уведомление менеджера
- сохранение заявок в таблицу

## Использовано
- Python
- aiogram
- Telegram Bot API
- Google Sheets API
